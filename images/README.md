# 图片说明

请将你的微信支付收款码图片放到这个目录，并重命名为 `wechat.png`

## 使用方法

1. 将你的微信支付收款码保存为图片
2. 复制到 `images/` 目录
3. 重命名为 `wechat.png`
4. 提交到 Git 仓库

## 示例

```bash
cp ~/Downloads/wechat-pay.png images/wechat.png
git add images/wechat.png
git commit -m "添加微信支付收款码"
```

## 注意事项

- 建议图片尺寸：200x200 像素
- 支持格式：PNG、JPG
- 文件大小：< 100KB
